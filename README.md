Transaction Reconciliation and Validation Workflow using Excel

This project demonstrates a practical reconciliation workflow using Excel to compare and validate transactions between Bank Statement and Cash Book datasets.

Key tasks performed:

Transaction matching using VLOOKUP and XLOOKUP
Missing transaction identification
Debit/Credit mismatch detection
Duplicate transaction analysis using COUNTIF
Pivot Table reporting and transaction summaries
Conditional Formatting for exception highlighting
Reconciliation KPI summary generation
